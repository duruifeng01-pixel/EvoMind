# 04 安卓前端 UI/UX 与模块开发

## 1. 信息架构（底部5 Tab）

- 首页（卡片流+每日一问+任务）
- 信息源（导入/管理/冷数据）
- 语料库（检索/分类/导出）
- 小组（匿名互助/成果区）
- 我的（订阅/支付记录/设置）

## 2. 关键页面

- 登录注册页：手机号验证码优先，微信快捷入口同屏展示
- 引导页：5步蒙层，支持跳过
- OCR导入页：上传截图→候选识别→一键导入（≤3步）
- 卡片详情页：导读、脑图入口、来源跳转、收藏分享
- 订阅页：算力成本、订阅费、成本占比显著展示

## 3. 交互规范

- 主要操作3步内完成
- 脑图支持双指缩放/点击展开/长按段落操作
- 完成关键操作触发轻震动与Toast反馈
- 所有AI输出附“AI生成，仅供参考”标签

## 4. 无障碍与适配

- 字号：小/中/大/超大
- 模式：深色/浅色/跟随系统
- 机型：华为、小米、OPPO、vivo、荣耀主流机型
- 系统：Android 10+
