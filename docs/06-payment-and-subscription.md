# 06 支付体系与透明订阅模式

## 1. 计费公式

- 实时算力成本 = Σ(指标用量 × 指标单价)
- 预估算力成本（展示）= ceil(实时算力成本)
- 订阅费 = 预估算力成本 × 2

## 2. 展示规则（订阅页）

- 预估算力成本：XX元/周(月)
- 订阅费：XX元/周(月)
- 成本构成：80%算力 + 20%运营

## 3. 套餐

- 基础：信息源≤20，讨论≤5/天，无冲突标记
- 进阶：信息源≤50，讨论≤20/天，Agent≤3/周
- 定制：按需实时估算

## 4. 支付流程

1. 选套餐
2. 确认订单（展示成本明细）
3. 选微信/支付宝
4. 调起SDK支付
5. 回调验签更新权益

## 5. 退款

- 非功能故障不支持全额退款
- 功能故障按剩余有效期比例退款
- 帮助与反馈提交工单，人工审核原路退回
