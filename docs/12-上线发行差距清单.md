# 12 上线发行差距清单（当前版本 vs 正式版）

## 已完成（可联调演示）

- 后端演示接口：认证、引导、信息源、认知、讨论、任务、订单、退款、隐私工单。
- 安卓端演示 UI：五Tab导航、中文文案、高级感深色渐变视觉。
- 部署骨架：Dockerfile + docker-compose + SQL 初始化脚本。

## 必须完成后才可正式上线

1. **服务端持久化**
   - 将 InMemoryStore 替换为 MySQL/Redis 实现。
   - 增加幂等、事务、审计日志。

2. **真实第三方SDK接入（国内）**
   - OCR：百度/阿里云 OCR。
   - 语音：讯飞语音识别。
   - AIGC：文心/通义/星火模型路由。
   - 支付：微信支付与支付宝 SDK + 回调验签。

3. **安卓端完整业务页面**
   - 登录注册/找回密码/微信登录页。
   - 信息源导入页（截图识别、手动导入、管理）。
   - 认知卡片页、脑图页、每日一问页、挑战任务页、支付订阅页。
   - Room 本地数据库与离线同步。

4. **安全与合规**
   - 隐私政策、用户协议、AIGC说明、权限说明。
   - 内容审核链路接入。
   - 数据导出与注销的真实执行流程。

5. **测试与发布**
   - 自动化测试稳定通过。
   - 主流机型兼容测试（华为/小米/OPPO/vivo/荣耀）。
   - 打包签名 APK/AAB 并完成应用市场材料提交。
