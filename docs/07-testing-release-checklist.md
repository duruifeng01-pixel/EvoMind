# 07 全功能联调、安卓兼容性测试与上架清单

## 1. 联调清单

- 账号登录/注册/注销全链路
- OCR导入→卡片→讨论→任务→分享闭环
- 订阅购买/续费/退款链路
- 本地离线读写与在线同步冲突处理

## 2. 测试类型

- 功能测试：核心流程100%覆盖
- 性能测试：启动、渲染、接口时延、内存
- 兼容测试：Android 10~14，主流机型矩阵
- 安全测试：越权、注入、弱口令、数据泄露
- 合规测试：AI标注、内容审核、隐私权限

## 3. 上架材料

- 隐私政策、用户协议、AIGC说明
- 第三方SDK清单与用途说明
- 权限申请说明与触发场景
- 软著、ICP/备案（如需）与公司资质

## 4. 发布流程

1) 提测包（internal）
2) 灰度包（10%）
3) 全量包（100%）
4) 监控回归与热修复
